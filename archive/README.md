# Archive

Retired code kept for reference. Not used by the active stack.

## Contents

### `actuator_mqtt_direct.py`

The original actuator that controlled smart lights and sirens directly via MQTT
(paho-mqtt) and played TTS via Snapcast FIFO pipes. Replaced in March 2025 by
the HA bridge actuator, which delegates all physical device control to Home
Assistant via its REST API.

Key features of this version (reusable if needed):
- `LightController` — MQTT RGB light control with color presets
- `AlarmController` — MQTT siren/alarm control (Zigbee sirens)
- `TTSPlayer` — Snapcast FIFO-based TTS with WAV playback and cooldowns
- User-configurable shell scripts per alert level (`SCRIPT_*` env vars)

### `audio/`

Pre-generated TTS WAV files used by the old actuator's Snapcast FIFO playback.
Generated by `generate_audio.py` using OpenAI TTS API.

### `generate_audio.py`

Script to generate the WAV files above using OpenAI's TTS API. Requires
`OPENAI_API_KEY` env var.
